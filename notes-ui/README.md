# Notes App UI — React Native (Expo)

A modern and responsive Notes App UI built using React Native with Expo.

## Features Implemented

### Notes Listing Screen

* Notes displayed using `FlatList`
* Search/filter functionality using `TextInput`
* Each note wrapped inside `Pressable`
* Responsive card layouts
* Dark/Light mode support
* Category filtering UI
* Floating action button
* Clean typography and spacing system

### Note Editor Screen

* Title input field
* Multiline note content editor
* `KeyboardAvoidingView` support
* Responsive writing layout
* Save and Back buttons using `Pressable`
* Comfortable long-form writing experience

## React Native Components Used

* `FlatList`
* `Text`
* `View`
* `TextInput`
* `Pressable`
* `Switch`
* `SafeAreaView`
* `KeyboardAvoidingView`
* `ImageBackground`

## Hooks Used

* `useState`
* `useMemo`
* `useColorScheme`
* `useWindowDimensions`
* Custom Hooks:

  * `useTheme`
  * `useNotes`

## Additional Improvements

* Custom theme system with reusable color tokens
* Responsive tablet + mobile layouts
* Context API based state management
* Modular reusable component structure
* Optimized filtering with memoization
* Modern UI inspired by productivity apps

## Styling

* Fully styled using `StyleSheet.create()`
* Dynamic theme-based styling
* Reusable spacing and typography constants
* Responsive layout handling for different screen sizes

## Tech Stack

* React Native
* Expo
* Context API
* Expo Vector Icons
